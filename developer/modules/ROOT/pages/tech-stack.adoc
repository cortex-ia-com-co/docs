= Stack Tecnológico

El stack tecnológico de UnoSportClub está basado en tecnologías open source modernas y probadas. Esta sección detalla las tecnologías utilizadas en frontend, backend, datos y cloud.

== Frontend

El frontend de UnoSportClub está construido con tecnologías modernas y reactivas.

=== Framework Principal

* *Angular*: Framework principal para aplicaciones web
* *TypeScript*: Superset de JavaScript con tipado estático
* *HTML5/CSS3*: Estándares web modernos

=== Estilos y UI

* *Tailwind CSS*: Framework CSS utility-first
* *SCSS/SASS*: Preprocesador CSS
* *Responsive Design*: Mobile-first approach
* *Progressive Web Apps (PWA)*: Capacidades nativas en web

=== Herramientas Frontend

* *Angular CLI*: Herramientas de línea de comandos
* *RxJS*: Programación reactiva
* *Angular Material*: Componentes de UI (opcional)

== Backend

El backend está construido con Node.js y Express.

=== Runtime y Framework

* *Node.js*: Runtime de JavaScript en el servidor
* *Express*: Framework web minimalista
* *Firebase Functions*: Funciones serverless

=== APIs y Comunicación

* *RESTful APIs*: Arquitectura REST
* *JSON*: Formato de intercambio de datos
* *Middleware*: Procesamiento de peticiones

=== Rutas y Endpoints

El backend está estructurado en:

* `functions/routes/` - Rutas de la API
* `functions/services/` - Servicios de negocio
* `functions/middleware/` - Middleware personalizado

== Base de Datos y Cloud

UnoSportClub utiliza Firebase como plataforma cloud.

=== Firebase Services

* *Firebase Auth*: Autenticación y autorización
* *Firebase Functions*: Lógica del servidor
* *Firebase Hosting*: Hosting estático

=== Características

* Escalabilidad automática
* Autenticación integrada
* Hosting global

== DevOps y Herramientas

=== Control de Versiones

* *Git*: Sistema de control de versiones
* *GitHub*: Repositorio y colaboración
* *Git Submodules*: Gestión de dependencias

=== CI/CD

* *GitHub Actions*: Automatización de workflows
* *Testing automatizado*: Tests unitarios e integración
* *Deployment automatizado*: Despliegue continuo

=== Herramientas de Desarrollo

* *npm*: Gestor de paquetes Node.js
* *Angular CLI*: Herramientas de desarrollo
* *Firebase CLI*: Herramientas de Firebase
