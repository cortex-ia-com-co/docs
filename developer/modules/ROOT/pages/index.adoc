== Arquitectura General

UnoSportClub es un sistema de gestión de reservas de canchas deportivas construido con:

* *Frontend*: Angular (aplicación web)
* *Backend*: Node.js con Express (API REST)
* *Base de datos*: PostgreSQL
* *Autenticación*: Firebase Auth
* *Funciones*: Firebase Functions (`api/index.js`)
